# Assets Directory

This directory contains visual assets for the WebStats project.

## Structure

```
assets/
├── screenshots/    # Application screenshots
├── icons/         # Icon files
└── branding/      # Logo and brand assets
```

## Screenshots Needed

1. **Status Page** - Main dashboard showing service statuses
2. **History Page** - Monitoring history and trends
3. **Incidents Page** - Incident tracking and resolution
4. **Mobile View** - Responsive design on mobile devices
5. **Dark Theme** - Terminal-core dark mode interface

## Image Specifications

- **Format**: PNG or JPEG
- **Size**: 1200x800 pixels minimum for screenshots
- **Quality**: High resolution for clear viewing
- **Naming**: Descriptive names (e.g., `status-dashboard.png`)

## Adding Screenshots

To add screenshots to the README:

1. Place images in the appropriate subdirectory
2. Reference them in README using relative paths
3. Use alt text for accessibility
4. Keep file sizes reasonable (< 500KB each)
